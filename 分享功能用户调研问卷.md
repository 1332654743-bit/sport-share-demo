## 二、场景A：日常快速打卡分享

> 想象你刚完成一次普通训练（常规跑步/骑行/健身），想快速发个朋友圈/运动群记录一下。

**Q3. 日常打卡时，你通常使用哪种分享方式？**（单选）

- [ ] A. 运动报告长图（直接截图或一键生成）
- [ ] B. 水印海报（选个模板贴数据）
- [ ] C. 轨迹视频
- [ ] D. 直接截图运动App页面

**Q4. 日常打卡场景下，你愿意花多长时间编辑？**（单选）

- [ ] A. 0秒，希望一键生成直接发
- [ ] B. 5-10秒，最多选一下模板
- [ ] C. 10-30秒，调整展示的数据项
- [ ] D. 30秒以上也可以接受

**Q5. 日常打卡时，运动报告里哪些编辑你会用？**（可多选）

- [ ] A. 不编辑，默认就好
- [ ] B. 选择/取消部分数据项（比如隐藏卡路里）
- [ ] C. 选择展示哪些图表
- [ ] D. 调整地图/轨迹样式

## 三、场景B：精心制作一次分享

> 想象你完成了一次特殊运动（首马/PB/风景绝美的越野/旅行骑行），想精心做一条内容发出去。

**Q6. 这种场景下，你更想用哪种方式分享？**（单选）

- [ ] A. 精美的水印海报（图片+数据贴纸，发朋友圈/社交平台）
- [ ] B. 轨迹视频（运动轨迹动画+照片+音乐，发视频号/抖音）
- [ ] C. 详细的运动报告（完整数据长图）

**Q7. 精心制作场景下，你愿意花多长时间编辑？**（单选）

- [ ] A. 30秒以内（快速调几项就好）

- [ ] C. 2-5分钟（仔细打磨每个细节）
- [ ] D. 5分钟以上也没问题，只要效果好

**Q8. 精心制作水印海报时，你最在意哪些编辑功能？**（最多选3项）

- [ ] A. 模板风格（3D/动态模糊/马赛克等特效）

- [ ] D. 选择展示哪些数据/图表水印
- [ ] E. 颜色编辑（配色统一/个性化）
- [ ] F. 以上都不太需要，默认效果就够了

**Q9. 精心制作轨迹视频时，你最在意哪些编辑功能？**（最多选3项）

- [ ] A. 添加运动中的照片到视频里
- [ ] B. 地图样式选择（经典/卫星/暗黑等）
- [ ] C. 轨迹样式效果（颜色/动画）
- [ ] D. 背景音乐选择
- [ ] E. 视频时长/节奏控制
- [ ] F. 以上都不太需要，自动生成就好

## 四、功能深度 vs 简洁度

**Q10. 对于「水印海报」你更倾向？**（单选）

- [ ] A. 功能丰富：模板多、背景可换可裁、颜色随意调——哪怕操作步骤多一些
- [ ] B. 中间路线：提供几套精美模板，一键套用，只开放关键编辑（换背景+选数据）
- [ ] C. 极简一键：AI 自动匹配最佳模板+配色，不用编辑直接出图

**Q11. 对于「轨迹视频」你更倾向？**（单选）

- [ ] A. 功能丰富：地图/轨迹/照片/音乐都能调——哪怕编辑要花几分钟
- [ ] B. 中间路线：选一个风格预设，只能加照片和换音乐
- [ ] C. 极简一键：自动生成视频，顶多选个音乐就发出

**Q12. 对于「运动报告」你更倾向？**（单选）

- [ ] A. 高度自定义：每个数据项/图表都可选择展示或隐藏
- [ ] B. 智能推荐：根据运动类型自动推荐最适合展示的数据组合，可微调
- [ ] C. 固定模板：不用选，默认展示最关键的数据就好
